# 📝 Task Manager

A simple and responsive **Task Manager Web Application** built using **HTML, CSS, and JavaScript**.

This project allows users to create, update, delete, and complete daily tasks. It uses **Local Storage** to save tasks permanently in the browser.

## 🚀 Features

- ➕ Add new tasks
- ✏️ Edit tasks
- 🗑️ Delete tasks
- ✅ Mark tasks as completed
- 💾 Saves data using Local Storage
- 🔄 Tasks remain after refreshing the page
- 📱 Responsive user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Browser Local Storage
